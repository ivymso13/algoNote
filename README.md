# 알고리즘 노트

알고리즘 학습 내용을 정리하고 관리할 수 있는 웹 기반 노트 애플리케이션입니다.

## 프로젝트 개요

- **이름**: 알고리즘 노트
- **목적**: 알고리즘 코드, 설명, 복잡도 등을 체계적으로 정리하고 관리
- **기술**: HTML, CSS, JavaScript (Vanilla JS)
- **데이터 저장**: LocalStorage

## 주요 기능

### ✅ 완료된 기능

- **노트 관리**
  - 알고리즘 노트 작성, 수정, 삭제
  - 카테고리별 분류 (정렬, 탐색 등)
  - 제목, 설명, 코드, 복잡도, 태그, 링크 관리

- **카테고리 관리**
  - 카테고리 추가, 수정, 삭제
  - 카테고리별 노트 개수 표시
  - 카테고리 접기/펼치기 기능

- **검색 및 필터링**
  - 실시간 검색 (제목, 설명, 태그)
  - 카테고리별 필터링

- **코드 하이라이팅**
  - Prism.js를 이용한 문법 강조
  - Python, Java, C++, JavaScript 지원
  - 코드 복사 기능

- **마크다운 지원**
  - 설명 작성 시 마크다운 문법 사용
  - 실시간 렌더링

- **다크 모드**
  - 라이트/다크 모드 전환
  - 설정 자동 저장

- **반응형 디자인**
  - 모바일/태블릿/데스크톱 대응

## 데이터 구조

### 노트 객체
```json
{
  "id": "고유ID",
  "title": "알고리즘 제목",
  "category": "카테고리ID",
  "description": "마크다운 설명",
  "code": "알고리즘 코드",
  "language": "python",
  "timeComplexity": "O(n log n)",
  "spaceComplexity": "O(log n)",
  "tags": ["태그1", "태그2"],
  "links": ["관련링크1", "관련링크2"],
  "createdAt": "생성일시",
  "updatedAt": "수정일시"
}
```

### 저장 방식
- **LocalStorage 키**
  - `algorithmNotes`: 노트 데이터 배열
  - `categories`: 카테고리 목록
  - `collapsedCategories`: 접힌 카테고리 ID 목록
  - `theme`: 테마 설정 (light/dark)

## 사용 방법

1. **GitHub Pages 배포**
   - 이 저장소를 GitHub에 푸시
   - Settings > Pages에서 GitHub Pages 활성화
   - `main` 브랜치 선택

2. **로컬에서 실행**
   - `index.html` 파일을 브라우저에서 직접 열기

3. **노트 작성**
   - 좌측 하단 "+ 새 노트" 버튼 클릭
   - 필요한 정보 입력 후 저장

4. **카테고리 관리**
   - 좌측 하단 "⚙️ 카테고리 관리" 버튼 클릭
   - 카테고리 추가/수정/삭제 가능
